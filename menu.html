Serious Menu

Tips & Tricks

  • Wyłącz podświetlenie po najechaniu myszką
  • Usuń podgląd po kliknięciu
  • Kliknięcie otwiera zdjęcie w nowym oknie
  • Wyłącz pauzę

Dostosowanie wysokości kontenera:

#three-container {
  height: 200px;
  width: 100%;
  display: block;
}

Konfiguracja w skrypcie:

const CONFIG = {
  defaultSpacing: 1.5,
  ...
}

Tylko pierwszy skrypt jest wymagany, plus deklaracja w <head> i jedna klasa CSS.
